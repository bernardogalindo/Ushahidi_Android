This project has three external library dependencies:

1. Refer to how to reference a library project from 
http://developer.android.com/guide/developing/projects/projects-eclipse.html#ReferencingLibraryProject 
then reference 'ActionBarLibrary' library project in CoreV3.

2. Obtain a copy of actionbarsherlock-plugin-maps-3.4.1.jar from 
http://actionbarsherlock.com/download.html
Once you have a copy of the package, place it into CoreV3/libs directory.

3. Download a copy of 'actionbarsherlock-plugin-preference-3.4.1.jar' from 
http://actionbarsherlock.com/download.html  once downloaded, place it 
into CoreV3/libs directory.
